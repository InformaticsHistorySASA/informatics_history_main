프로그래밍 언어의 역사

1950-1960년대

  • 1957년: Fortran (존 배커스)

    과학 및 수치 계산용 최초의 고급 프로그래밍 언어.

  • 1958년: Lisp (존 매카시)

    인공지능 연구와 함수형 프로그래밍의 기초가 된 언어.

  • 1959년: COBOL

    기업과 비즈니스 데이터 처리에 널리 사용된 최초의 상업용 언어.

1970년대

  • 1972년: C (데니스 리치)

    운영 체제 및 시스템 소프트웨어 개발에 사용되며, 현대 많은 언어(C++, Java 등)의 기반이 된 언어.

  • 1978년: SQL

    데이터베이스 관리와 쿼리 언어의 표준을 세운 언어.

1980-1990년대

  • 1983년: C++ (비야네 스트롭스트룹)

    C에 객체 지향 개념을 추가한 언어로, 고성능 소프트웨어 및 게임 개발에 널리 사용됨.

  • 1991년: Python (귀도 반 로섬)

    코드 가독성, 생산성 높은 문법으로 데이터 과학, 웹 개발, 인공지능 등 여러 분야에서 인기 있는 언어.

  • 1995년: Java (제임스 고슬링)

    플랫폼 독립적 언어로, 엔터프라이즈 소프트웨어, 안드로이드 앱 개발 등에 광범위하게 사용됨.

  • 1995년: JavaScript (브렌던 아이크)

    웹 프론트엔드 개발의 표준 언어로, 브라우저에서 동작하며 현대 웹 애플리케이션 개발에 필수적.

2000년대 이후

  • 2000년: C#

    마이크로소프트의 .NET 프레임워크 언어로, 데스크탑 애플리케이션, 웹 개발, 게임 개발(Unity) 등에 사용됨.

  • 2009년: Go (Golang)

    구글에서 개발한 언어로, 간결한 문법과 높은 성능을 제공하며 서버 및 클라우드 네이티브 애플리케이션 개발에 사용됨.

  • 2014년: Swift

    애플의 iOS, macOS 앱 개발을 위한 언어로, 안전성과 성능을 강조.

  • 2016년: TypeScript

    JavaScript에 정적 타입을 추가하여 대규모 애플리케이션 개발에 최적화된 언어.
