* 전문가 시스템(expert system)

: 특수한 영역에서 인간 전문가의 의사 결정 능력을 흉내내는 컴퓨터 시스템


인간의 지적 활동과 경험으로 축적된 전문가 지식 + 전문가에 의해 정의된 추론 규칙

규칙으로 표현되는 지식을 통해 추론 -> 복잡한 문제를 해결하도록 설계됨

궁극적으로 전문적인 지식을 체계적으로 적용하는 인간의 사고과정 흉내


* 전문가 시스템의 필수 요소 = 지식 베이스(데이터베이스) + 추론 기구(지식 베이스에 기초)


지식 베이스(Knowledge base)

- 특수한 문제 영역에 대한 구체적 사실과 규칙 제공

- 지식 추가 가능, 자동 학습 불가능

- 추론 과정에서 규칙으로부터 유도된 사실은 동적으로 추가 가능


추론 엔진(Inference engine)

- 전문가 시스템이 결론 도출하는 추론 능력 제공

- 문제 해결을 위해 적용할 사실과 규칙 결정

- 지식 베이스와 독립적인 추론 기관의 구조


* 전문가 시스템의 툴 = 사용자 인터페이스 + 설명 기능


사용자 인터페이스(user interface)

- 사용자 질의와 정보 입력 등 시스템과 상호작용


설명 기능(description function)

- 시스템 자신이 도출한 결론에 대해 설명하거나 정당화 -> 개발자 시스템 작동 검사


* 전문가 시스템의 예시

최초의 전문가 시스템 DENDRAL(화학 전문가 시스템)

화합물의 구성요소와 화합물 샘플로부터 질량 스펙트로그램 -> 화합물 분자구조 결정

* 전문가 시스템의 장점

- IF-THEN 규칙을 사용해 인간 전문가의 지식을 자연스럽게 표현 가능하다

- IF-THEN 구조의 지식은 독립적이고 설명 없이 직관적으로 이해할 수 있다

- 지식베이스와 추론 엔진이 분리되어 다른 영역에 쉽게 적용 가능

- 확신도를 이용해 불완전, 불확실 지식 표현도 가능


* 전문가 시스템 단점

- 과거의 사례로부터 지식의 학습이 불가능

- 스스로 규칙 변경, 추가 어려움 -> 지식 공학자가 지식 베이스 수정 유지 필요

- 탐색 비효율적

- 규칙이 많아지면 상호 관계 불명확하고 규칙의 유지 보수가 어려움
