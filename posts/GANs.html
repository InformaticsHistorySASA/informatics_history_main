Generative Adversarial Networks, 2014년 Ian Goodfellow가 제안한 생성적 적대 신경망, AI 모델, 두 개의 신경망 서로 경쟁해 데이터 생성 방식


생성자(새로운 데이터 만듦), 판별자(데이터 실제, 가짜 구분) 2개의 신경망으로 구성되어 과정 반복으로 생성자가 현실적인 데이터 생성 가능하다.


이미지 생성, 비디오 생성, 텍스트 데이터 생성 등에 이용되고 현실적 이미지 생성에 뛰어나다. 비지도 학습의 사례로, 라벨 없는 데이터 학습 가능하다.


딥페이크 기술이나 이미지 복원 등의 작업에 사용되어 정교하고 현실적인 데이터 생성 능력 가진다.
